Integration of a Search Box

The following provides information on how to integrate a search box on any web page that calls the normal YaCy search window.

Simply use the following code:


<form method="get" accept-charset="UTF-8" action="http://#[myaddress]#/yacysearch.html">
  <div style="display:inline-block;text-align:center;padding:5px;background-color:#eee;border:1px solid #ccc;border-radius:5px;margin-right:5px;">
    <div style="display:inline-block;font-family:Arial,Helvetica,sans-serif;padding-top:3px;padding-right:5px;">
      MySearch
    </div>
    <input type="text" name="query" maxlength="80"
           style="display:inline-block;width:300px;"/>
    <input type="hidden" name="verify" value="cacheonly"/>
    <input type="hidden" name="maximumRecords" value="10"/>
    <input type="hidden" name="meanCount" value="5"/>
    <input type="hidden" name="resource" value="local"/>
    <input type="hidden" name="urlmaskfilter" value=".*"/>
    <input type="hidden" name="prefermaskfilter"/>
    <input type="hidden" name="display" value="2"/>
    <input type="hidden" name="nav" value="all"/>
    <div style="display:inline-block;padding-top:1px;">
      <input type="submit" name="Enter" value="Search"/>
    </div>
  </div>
</form>
This would look like:
MySearch
This does not use a style sheet file to make the integration into another web page with a different style sheet easier. You would need to change the following items:
  • Replace the given colors #eee (box background) and #ccc (box border)
  • Replace the word MySearch with your own message
