RWI Ranking Configuration

The document ranking influences the order of the search result entities. A ranking is computed using a number of attributes from the documents that match with the search word. The attributes are first normalized over all search results and then the normalized attribut is multiplied with the ranking coefficient computed from this list. The ranking coefficient grows exponentially with the ranking levels given in the following table. If you increase a single value by one, then the strength of the parameter doubles.
